# Run Kitty Run

---

A simple 2D platformer made using Pygame Zero.

---

## Game Idea

You control a kitty that keeps moving while the world scrolls to the left.
Avoid getting pushed off the screen and defeat enemies to win.

---

## Controls

Left Arrow      -> Move left
Right Arrow     -> Move right
Space           -> Jump / Start / Restart

---

## Objective

* Jump on enemies to defeat them
* Kill 2 enemies to unlock the win condition
* Reach the right side of the screen to win
* If you touch the left border, you lose

---

## Sound

The game includes:

* Background music
* Jump sound
* Win sound
* Lose sound

You can turn sound on/off in the code:
sound_on = True

## How to Run

1. Install Pygame Zero:
   pip install pgzero

2. Run the game:
   pgzrun main.py
